max-allowed-packet的问题

发表于 2017-08-17 | 分类于 Mysql

前言

近期，因启动项目有个批量插入的sql结果太大，超过了mysql自带的缓存，报了这个错误

修改： 定位到mysql的安装目录下面，然后修改my.ini 的max_allowed_packet = 8M
默认是1M
